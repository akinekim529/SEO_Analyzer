### 🚀 How to Use

#### Quick Start:
```bash
# 1. Install dependencies
pip3 install -r requirements.txt

# 2. Add OpenAI API key to .env file
# Edit .env and replace: OPENAI_API_KEY=your_actual_api_key_here

# 3. Run analysis
python3 seo_analyzer.py
```

#### Demo Mode (No API Key Required):
```bash
python3 demo.py
```
